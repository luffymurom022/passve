🔗 Mở Supabase
📍 Cách chạy SQL trong Supabase:
1. Vào app.supabase.com → chọn project của bạn
2. Click SQL Editor ở menu trái
3. Click + New query
4. Paste nội dung file .sql vào → click ▶ Run
5. Lặp lại cho từng file theo thứ tự bên trên
