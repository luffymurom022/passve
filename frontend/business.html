API Documentation

Tài liệu tích hợp SafePass Business API

Base URL: https://your-repl.replit.app  |  Authentication: X-Api-Key: <your_key>
🛡️ Escrow API
POST /api/business/escrow/create Tạo giao dịch escrow mới
Headers: X-Api-Key: sk_live_xxx
Body JSON:
{ "title": "Mua tài khoản Netflix Premium", "description": "Tài khoản 1 tháng", "amount": 150000, "buyer_email": "buyer@email.com", "seller_email": "seller@email.com", "ref": "order_001", "category": "account", "expires_hours": 48 }
POST /api/business/escrow/:id/release Giải ngân cho seller
Headers: X-Api-Key: sk_live_xxx
Body JSON:
{ "note": "Buyer đã xác nhận nhận hàng" }
POST /api/business/escrow/:id/refund Hoàn tiền cho buyer
Body JSON:
{ "reason": "Seller không giao hàng đúng hạn" }
GET /api/business/escrow/:id Lấy trạng thái escrow
Response:
{ "id": "uuid", "ref": "order_001", "status": "funded", "amount": 150000, "created_at": "2026-06-20T..." }
⭐ Trust & Fraud API
GET /api/business/trust/:phone Trust score của người dùng SafePass
Response:
{ "phone": "0901234567", "trust_score": 87, "level": "trusted", "total_orders": 12, "completed_orders": 11, "avg_rating": 4.8, "is_verified": true }
POST /api/business/fraud-check Kiểm tra rủi ro giao dịch
Body:
{ "phone": "0901234567", "amount": 5000000 }
Response:
{ "risk_level": "low", "risk_score": 12, "flags": [], "recommendation": "APPROVE" }

EventMô tả
